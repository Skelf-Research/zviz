# Development Setup

Set up your development environment.

## Prerequisites

- Zig 0.15.0+
- Git
- Linux (for testing)

## Setup

```bash
git clone https://github.com/zviz/zviz.git
cd zviz
zig build
zig build test
```

## IDE Setup

### VS Code

Install the Zig extension.

### Neovim

Use `zls` language server.

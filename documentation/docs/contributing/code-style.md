# Code Style

ZigViz code conventions.

## Formatting

```bash
zig fmt src/
```

## Naming

- Functions: `camelCase`
- Types: `PascalCase`
- Constants: `SCREAMING_SNAKE_CASE`

## Documentation

Use doc comments:

```zig
/// Brief description.
///
/// Detailed description.
pub fn example() void {}
```
